## 1. Alertas e Avisos (Asides)

O Starlight possui componentes embutidos para destacar informações críticas.

<Aside>
  Um alerta padrão (nota). Útil para informações gerais.
</Aside>

<Aside type="tip">
  **Dica:** Você pode usar Markdown padrão, como **negrito** ou *itálico*, dentro destes blocos.
</Aside>

<Aside type="caution" title="Atenção Personalizada">
  Você pode alterar o título padrão do alerta usando a prop `title`.
</Aside>

<Aside type="danger">
  Cuidado! Algo crítico pode acontecer aqui.
</Aside>

## 5. Passos (Steps)

Perfeito para tutoriais sequenciais. O Starlight cuida da numeração e do design.

<Steps>

1. **Instale o projeto**

    Execute o comando de inicialização no seu terminal.

2. **Configure o ambiente**

    Edite o arquivo `astro.config.mjs`.

3. **Deploy**

    Envie para produção!

</Steps>
